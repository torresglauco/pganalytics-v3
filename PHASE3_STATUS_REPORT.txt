1. ENTERPRISE AUTHENTICATION (8 ENDPOINTS)
   ✅ LDAP/Active Directory
      - TLS support, group-to-role mapping, service accounts
      - Endpoint: POST /api/v1/auth/ldap/login
   
   ✅ SAML 2.0 Single Sign-On
      - Metadata generation, assertion parsing, signature verification
      - Endpoints: GET /api/v1/auth/saml/metadata, POST /api/v1/auth/saml/acs
   
   ✅ OAuth 2.0/OIDC (Google, Azure, GitHub, custom)
      - Authorization code flow, PKCE-ready
      - Endpoints: GET /api/v1/auth/oauth/:provider/login, POST /api/v1/auth/oauth/callback
   
   ✅ Multi-Factor Authentication
      - TOTP (Time-based OTP), SMS delivery, backup codes
      - Endpoints: POST /api/v1/users/mfa/setup, POST /api/v1/auth/mfa/challenge
