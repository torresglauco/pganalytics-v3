Host status detection logic:
- Query collectors.last_seen timestamp
- Compare with current time minus threshold (default 5 minutes)
- If last_seen > threshold: status = "up", is_healthy = true
- If last_seen < threshold: status = "down", is_healthy = false
- If last_seen is NULL: status = "unknown"

Host inventory sources:
- OS info: /etc/os-release (Linux), uname syscall
- CPU: /proc/cpuinfo, sysconf(_SC_NPROCESSORS_ONLN)
- Memory: /proc/meminfo (already in sysstat)
- PostgreSQL config: pg_settings view
