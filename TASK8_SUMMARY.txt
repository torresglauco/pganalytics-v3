┌─ SECURITY IMPROVEMENTS ─────────────────────────────────────────────────────┐
│                                                                             │
│ BEFORE (localStorage):                                                     │
│   ❌ XSS Attack Vector                                                     │
│      <script>alert(localStorage.getItem('auth_token'))</script>           │
│      → Attacker steals token and impersonates user                        │
│                                                                             │
│   ❌ CSRF Vulnerability                                                    │
│      No CSRF token validation required                                    │
│      → Attacker can make requests from another site                       │
│                                                                             │
│                                                                             │
│ AFTER (httpOnly cookies):                                                 │
│   ✅ XSS Protected                                                         │
│      • auth_token NOT accessible to JavaScript                            │
│      • Even if XSS occurs, attacker can't steal token                    │
│      • Cookies automatically sent by browser                              │
│                                                                             │
│   ✅ CSRF Protected                                                        │
│      • X-CSRF-Token header required for mutations                        │
│      • csrf_token extracted from non-httpOnly cookie                     │
│      • Attacker can't read csrf_token from another site                  │
│      • Request without valid csrf_token is rejected                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
