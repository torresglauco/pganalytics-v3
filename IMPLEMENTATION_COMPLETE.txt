TASK 2: FIRST-LOGIN PASSWORD CHANGE - COMPLETE ✅
================================================================================

PROBLEM SOLVED:
- New users created via /api/v1/auth/setup did not force password change
- Users could login with setup-provided passwords indefinitely
- No security enforcement for initial password setup

SOLUTION IMPLEMENTED:
1. Added password_changed Field to User
   - Default: false (password not changed)
   - Set to true when user changes password
   - Included in all user queries

2. New API Endpoint
   - GET /api/v1/auth/password-change-required
   - Returns: { password_change_required: boolean }
   - Requires authentication

3. Updated Password Change Flow
   - Modified handleChangePassword() to use UpdateUserPassword()
   - New UpdateUserPassword() method sets password_changed = true
   - Old ResetUserPassword() remains for admin actions

4. Database Schema
   - ALTER TABLE adds password_changed BOOLEAN DEFAULT false
   - CREATE INDEX for efficient queries
   - Handles existing users (defaults to false)

FLOW:
1. Admin creates user via setup endpoint
   → password_changed = false
2. User logs in with temporary password
   → Receive JWT tokens
3. Frontend calls password-change-required
   → Returns true
4. Frontend shows password change modal
   → User enters new password
5. Frontend calls change-password endpoint
   → password_changed = true
6. User can proceed to dashboard

EXAMPLE FLOW (REACT):
  useEffect(() => {
    checkPasswordChangeRequired();
  }, [accessToken]);

  async function checkPasswordChangeRequired() {
    const response = await fetch('/api/v1/auth/password-change-required', {
      headers: { Authorization: `Bearer ${accessToken}` }
    });
    const data = await response.json();
    if (data.password_change_required) {
      setShowPasswordChangeModal(true);
    }
  }

  async function handlePasswordChange(oldPassword, newPassword) {
    const response = await fetch('/api/v1/auth/change-password', {
      method: 'POST',
      headers: { Authorization: `Bearer ${accessToken}` },
      body: JSON.stringify({ old_password: oldPassword, new_password: newPassword })
    });
    if (response.ok) {
      setShowPasswordChangeModal(false);
    }
  }
