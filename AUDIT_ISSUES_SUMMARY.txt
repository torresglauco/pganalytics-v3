  [SECURITY] CORS Too Permissive
  ├─ File: backend/internal/api/middleware.go:260-261
  ├─ Problem: Allow-Origin: * + Allow-Credentials: true (violates CORS spec)
  ├─ Impact: CSRF vulnerability - any site can make authenticated requests
  ├─ Fix Time: 20 minutes
  └─ Fix: Whitelist specific domains

  [SECURITY] JWT Token in localStorage (XSS Vulnerable)
  ├─ Files: frontend/src/api/authApi.ts, frontend/src/services/api.ts
  ├─ Problem: Tokens stored in localStorage (vulnerable to XSS)
  ├─ Impact: Any injected script can steal auth tokens
  ├─ Fix Time: 2 hours
  └─ Fix: Use httpOnly, Secure, SameSite cookies

  [SECURITY] No Token Revocation/Blacklist
  ├─ File: backend/internal/auth/ (missing)
  ├─ Problem: Logout doesn't invalidate token (valid for 15 minutes)
  ├─ Impact: Compromised session remains valid after logout
  ├─ Fix Time: 3-4 hours
  └─ Fix: Implement Redis blacklist or DB-based invalidation

  [TESTS] E2E Tests - Silent Failures
  ├─ File: frontend/e2e/tests/05-user-management.spec.ts
  ├─ Problem: Wrong credentials, error catching hides failures
  ├─ Impact: Tests report passing but core features broken
  ├─ Fix Time: 1-2 hours
  └─ Fix: Use correct credentials, remove silent error catching

  [SECURITY] Hardcoded DB Passwords
  ├─ File: docker-compose.yml
  ├─ Problem: postgres:pganalytics (plain text in repository)
  ├─ Impact: Credentials exposed if repo is leaked
  ├─ Fix Time: 30 minutes
  └─ Fix: Use environment variables with strong secrets

  [SECURITY] Database SSL Disabled
  ├─ File: docker-compose.yml (sslmode=disable)
  ├─ Problem: Unencrypted DB connections expose credentials
  ├─ Impact: Man-in-the-middle attacks
  ├─ Fix Time: 20 minutes
  └─ Fix: Use sslmode=require or sslmode=verify-full

  [VALIDATION] Input Validation Inconsistent
  ├─ File: frontend/src/ (forms)
  ├─ Problem: Zod library available but not systematically used
  ├─ Impact: Frontend accepts invalid data
  ├─ Fix Time: 2-3 hours
  └─ Fix: Implement Zod schema for all form validation

  [TESTS] Collector Integration Tests Failing
  ├─ File: collector/tests/integration/
  ├─ Problem: 16/19 tests failing (connection, plugin issues)
  ├─ Impact: Collector not properly tested before deployment
  ├─ Fix Time: 90 minutes
