1. CalculateHostHealthScore(metrics *models.HostMetrics, weights models.HealthScoreWeights) int
   - CPU Score: 100 - (cpu_user + cpu_system + cpu_iowait), clamp to 0-100
   - Memory Score: 100 - memory_used_percent, clamp to 0-100
   - Disk Score: 100 - disk_used_percent, clamp to 0-100
   - Load Score:
     - If load_1m <= cpu_cores: score = 100
     - If load_1m > cpu_cores AND load_1m <= 2*cpu_cores: score = 50
     - If load_1m > 2*cpu_cores: score = 0
   - Weighted average: (cpu_score * weights.CPU) + (memory_score * weights.Memory) + (disk_score * weights.Disk) + (load_score * weights.LoadAverage)
   - Return int(score)

   - score >= 80: return "healthy"
   - score >= 60: return "degraded"
   - score >= 40: return "warning"
   - score < 40: return "critical"

3. CalculateHealthScoreWithDetails(metrics *models.HostMetrics, weights models.HealthScoreWeights) *models.HealthScore
   - Call CalculateHostHealthScore
   - Calculate individual component scores
   - Build calculation_details JSONB with contributing factors
   - Return complete HealthScore struct with status
